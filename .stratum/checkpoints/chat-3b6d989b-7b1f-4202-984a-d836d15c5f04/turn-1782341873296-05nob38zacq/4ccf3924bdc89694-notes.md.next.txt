# Project Notes

## Goals
Build a quantum computing + mechanics research archive website.

## Decisions
- Next.js 16 (App Router) with TypeScript and Tailwind CSS v4
- CSS-first Tailwind configuration (v4 style, no tailwind.config.js)
- Data layer uses TypeScript modules in src/data/
- All pages are statically generated at build time

## Useful Commands
- `npm run dev` — Start dev server at http://localhost:3000
- `npm run build` — Production build (no errors)
- `npm run start` — Start production server
- `npx next build` — Build with output details
